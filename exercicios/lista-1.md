# LISTA DE EXERCÍCIOS

[VOLTAR](/readme.md)

1. Hospital: Modelar um sistema de gerenciamento de um hospital, incluindo entidades como pacientes, médicos, enfermeiros, agendamento de consultas, histórico de saúde, medicamentos, etc.

2. Biblioteca: Modelar um sistema de gerenciamento de uma biblioteca, onde os alunos podem criar entidades como livros, clientes, empréstimos, devoluções, autores, gêneros literários, etc.

3. Escola: Modelar um sistema de gerenciamento escolar, incluindo entidades como alunos, professores, cursos, matrículas, notas, disciplinas, etc.

4. Restaurante: Modelar um sistema de gerenciamento de restaurante, com entidades como mesas, pedidos, cardápio, clientes, funcionários, etc.

5. E-commerce de Alimentos: Modelar um sistema de gerenciamento de um e-commerce de alimentos, com entidades como restaurantes parceiros, cardápios, pedidos, entregadores, avaliações, etc.

6. Rede Social: Modelar um sistema de gerenciamento de rede social que inclui perfis de usuários, amizades, postagens, comentários, curtidas, etc.

7. Agência de Viagens: Modelar um sistema de gerenciamento de agência de viagens que lida com reservas de voos, hotéis, passeios turísticos, clientes, etc.

8. Gestão de Projetos: Modelar um sistema de gerenciamento de projetos, com entidades como projetos, tarefas, equipes, prazos, recursos, etc.
