# Trabalho 1 - Modelagem de Dados

[VOLTAR](/readme.md)

Neste exercício, você terá a liberdade de escolher o tema para modelagem de dados de acordo com seus interesses ou experiências. Seu objetivo é criar um modelo conceitual e lógico de banco de dados para organizar e estruturar os dados de um determinado negócio, projeto ou cenário do mundo real.

## Instruções

Escolha um tema ou contexto para o seu projeto de modelagem de dados. Pode ser qualquer coisa, desde uma loja virtual até um sistema de gerenciamento de biblioteca, um aplicativo de rede social, uma plataforma de streaming de música, um sistema de gerenciamento de eventos, entre outros. Sinta-se à vontade para explorar seus interesses e escolher um tema que você considere interessante e desafiador.

## 1 - Resumo do Estudo de Caso

Escreva um breve resumo explicando o contexto do seu tema escolhido e o propósito do banco de dados modelado. Descreva como o banco de dados será utilizado e como ele beneficia o negócio ou organização relacionada ao tema.

## 2 - Modelo Conceitual

Crie um modelo conceitual de banco de dados para o seu tema escolhido. Identifique as entidades relevantes, seus atributos e os relacionamentos entre elas. Utilize diagrama de entidade-relacionamento (DER) do BRModelo ou outra ferramenta de modelagem de dados de sua preferência.

## 3 - Modelo Lógico

Com base no modelo conceitual, desenvolva um modelo lógico de banco de dados. Utilize diagrama de entidade-relacionamento (DER) do BRModelo ou outra ferramenta de modelagem de dados de sua preferência. Certifique-se de incluir todas as chaves primárias e estrangeiras necessárias para representar os relacionamentos entre as entidades no seu modelo lógico.

## 4 - Consultas

Liste pelo menos 10 consultas que seriam comuns nesse sistema, como "Encontre todos os produtos de uma determinada categoria" ou "Mostre todos os eventos agendados para uma determinada data".

## Entrega do Trabalho

O trabalho poderá ser feito individualmente ou em dupla. Envie um arquivo PDF contendo o resumo do estudo de caso, o modelo conceitual (DER), o modelo lógico (DER) e as 10 consultas elaboradas. Certifique-se de que o documento esteja organizado e claro para uma fácil compreensão.
